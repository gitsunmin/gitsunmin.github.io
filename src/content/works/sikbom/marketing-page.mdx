## 이 레포의 역할

식봄 서비스의 랜딩 페이지 및 마케팅 채널용 웹 서비스입니다. 잠재 고객(음식점 사장님, 유통사)에게 식봄 서비스를 소개하고 가입을 유도하는 퍼널의 시작점입니다. 빠른 초기 로딩과 SEO 최적화가 핵심 목표입니다.

## 담당 역할 & 기여

### SEO 최적화 및 메타 태그 관리

검색엔진 노출을 위한 메타 태그(OG, Twitter Card, JSON-LD 구조화 데이터)를 관리하고, 페이지별 타이틀·설명문을 최적화했습니다. B2B 서비스 특성상 "식자재 발주", "식자재 주문 앱" 등 타깃 키워드 중심으로 컨텐츠를 구성했습니다.

### 빠른 초기 로딩 최적화

PHP 서버사이드 렌더링을 활용해 JavaScript 없이도 전체 콘텐츠를 즉시 노출합니다. 이미지 lazy loading, 웹폰트 최적화(font-display: swap), 리소스 preload를 적용하여 LCP(Largest Contentful Paint)를 개선했습니다.

### 유지보수 및 콘텐츠 업데이트

신규 기능 출시, 프로모션 이벤트에 맞춰 랜딩 페이지 콘텐츠를 업데이트하고 A/B 테스트 결과를 반영했습니다.

## 기술 선택 이유

| 기술 | 선택 이유 |
|------|----------|
| **PHP** | 기존 인프라 재활용으로 별도 서버 구성 불필요, 서버사이드 렌더링으로 SEO에 유리 |
| **HTML/CSS/JavaScript** | 마케팅 페이지는 인터랙션이 단순 — 무거운 프레임워크 없이 빠른 로딩 우선 |

## 성과

- 검색엔진 최적화로 오가닉 유입 채널 확보
- 서버사이드 렌더링으로 JavaScript 비활성화 환경에서도 콘텐츠 노출 보장
