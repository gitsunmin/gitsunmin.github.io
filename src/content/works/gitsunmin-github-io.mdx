## 배경

단순한 이력서 페이지가 아닌, 다양한 실험적 UI·인터랙션을 직접 만들어보는 공간으로 활용하고 있습니다. Astro를 기반으로 블로그, 포트폴리오, TIL(Today I Learned) 등 다양한 콘텐츠를 운영하며, Three.js·View Transitions API 등 새로운 기술을 실험하는 사이드 프로젝트입니다.

## 주요 기능

- 블로그 (MDX 기반 포스팅, 코드 하이라이팅)
- TIL (Today I Learned) 섹션
- Works & Careers 포트폴리오
- 다크모드 / 폰트 크기 조절 접근성 기능
- View Transitions API로 페이지 전환 애니메이션
- Three.js 실험적 인터랙션
- RSS 피드 및 사이트맵 자동 생성

## 담당 역할 & 기여

- **전체 설계 및 개발** — 1인 개발, 기획·디자인·구현·배포 전담
- **Astro 기반 정적 사이트** — GitHub Pages 배포 최적화
- **React Island 아키텍처** — 필요한 부분만 React로 동적 처리 (Partial Hydration)
- **커스텀 UI 컴포넌트** — TechFilterBar, WorkCard, CareerCard 등 직접 구현
- **WebP 이미지 최적화** — Sharp 기반 빌드 타임 이미지 변환 스크립트

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 프레임워크 | Astro, React |
| 언어 | TypeScript |
| 스타일 | Tailwind CSS |
| 콘텐츠 | MDX |
| 3D | Three.js |
| 배포 | GitHub Pages |

### 기술 선택 이유

- **Astro**: 콘텐츠 중심 사이트에 최적화. Partial Hydration으로 JavaScript 최소화
- **Tailwind CSS v4**: 유틸리티 클래스 기반 빠른 UI 구현 및 다크모드 지원
- **View Transitions API**: 네이티브 브라우저 API를 활용한 부드러운 페이지 전환 (SPA 없이)
