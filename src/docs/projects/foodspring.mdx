## 목차

## 식봄 소개
 식봄은 식자재의 시장의 봄날을 기원하며 개발된 식자재 전용 B2B 커머스 플랫폼입니다. 식자재를 구입하는 구매자와 판매하는 유통사인 판매자를 고객으로하여 서비스가 운영되어지고 있습니다.

## 어떤 일을 하였는가?

### PHP to Next.js 전환 작업
식봄은 기존에 Code Igniter라는 PHP 프레임워크를 사용하고 있었으며, 프론트엔드는 Next.js, 백엔드는 Kotlin Spring Boot로 전환하는 프로젝트입니다.

#### 문제와 해결방법
- Problem: 이미 한창 운영이 되어지고 있는 서비스이며, 규모가 큰 프로젝트이기 때문에, 한 번에 모든 코드를 교체하는 것은 불가능에 가깝다. 어떻게 전환할 수 있을까?
- Solution: 코드를 제거하고 프론트엔드와 백엔드로 분리하고 각 분야별로 작업을 하도록 하였다.
