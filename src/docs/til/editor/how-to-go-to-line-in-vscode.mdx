# How to go to line in vscode

## Go to line hotkey

- Windows -> `Ctrl + G`
- Mac -> `control + G`

## Go to line with command palette

- Windows -> `Ctrl + Shift + P`
- Mac -> `command + Shift + P`

을 이용하여 command palette를 열고 `>`를 지우고 `:` 를 입력하면 line number를 입력할 수 있는 창이 나오고, 위 방법과 동일하게 이동이 가능합니다.
