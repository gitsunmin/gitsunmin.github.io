# Autocomplete commit message in Vscode with Copilot

1. vscode setting 창으로 진입 (CMD + ,)
2. `copilot` 검색
3. Github > Copilot: Enable의 Edit in settings.json 클릭
4. 값을 아래와 같이 수정
   ```json
   "github.copilot.enable": {
        ...
        "scminput": true
   }
   ```
5. vscode 재시작
6. commit message 가만히 멍 때리고 있으면 copilot이 동작한다.

## 참고

아직은 자동완성 수준인 것 같다.
