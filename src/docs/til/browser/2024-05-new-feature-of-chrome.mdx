# 2024 05, New Feature of Chrome

1. Emulate a focused page

- Open the DevTools
- Elements 탭 진입
- Elements > Styles 탭 진입
- Elements > Styles > Filter 입력 창 옆에 `:hov` 클릭
- 포커싱이 유지되었으면 하는 요소를 클릭
- Elements > Styles에 나타난 Emulate a focused page를 클릭


2. CSS 우선순위 확인하기

- Open the DevTools
- Elements 탭 진입
- Elements > Styles 탭 진입
- Element를 선택하여 해당 Element의 style을 확인
- CSS 선택자에 마우스를 올리면 우선순위가 나타남

3. 에러 메시지 AI로부터 설명 제공 받기

- Open the DevTools
- 오른쪽 상단의 톱니바퀴 클릭
- Console 섹션의 `Understanding console messages with AI`를 활성화
- Console 탭의 에러 메시지의 오른쪽 전구 모양 아이콘 클릭하면 AI가 에러 메시지를 설명해줌
