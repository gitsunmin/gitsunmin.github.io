# A11y
 a11y는 accessibiliy의 약자로서 a와 y의 사이에 11개의 글자가 있어서 a11y로 표기하고 있습니다. 이 a11y는 장애인, 비장애인의 구분 없이 웹 서비스를 이용할 수 있도록 하기 위한 지침인데, 한국에서는 `인식의 용이성`, `운용의 용이성`, `이해의 용이성`, `견고성` 4가지를 원칙으로 가지고 있습니다. 

a11y관련하여 지침이 잘 지켜지고 있는지 체크하는 라이브러리들도 많이 제공되니, 사용하면 좋을것 같습니다.

a11y 지침의 예)
- 이미지에 대한 대체 텍스트에 자세한 설명을 포함해야한다.
    - `<img src="..." alt="gitsunmin 카페 오픈 안내 그림 - 카페 gitsunmin이 오픈 하였습니다. 누구나 언제든 커피를 마실 수 있도록 준비 되어 있습니다.">`
- 저시력자, 노인 등 콘텐츠를 인식할 수 있도록, 배경과 텍스트의 명도 대비는 `4.5:1` 이상으로 제공해야합니다.
- 광과민성 발직을 예방하기 위해서 초당 3~50회 주기로 깜빡이는 콘텐츠를 제공하지 않아야한다.
