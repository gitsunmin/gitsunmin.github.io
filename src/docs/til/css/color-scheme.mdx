# Color Scheme

Css의 Color Scheme은 전체 컬러의 테마를 설정하는 속성입니다.

## Syntax

```css
color-scheme: light;
color-scheme: dark;
color-scheme: normal;
```

## 응용

prefers-color-scheme를 사용하여 사용자의 시스템 설정에 따라 테마를 변경할 수 있습니다.

```css
@media (prefers-color-scheme: dark) {
  form {
    color-scheme: dark;
  }
}
```
