# Placeholder Shown

input 요소의 placeholder가 보여지는 상태를 선택할 수 있는 pseudo-class입니다.

```css
/* placeholder-shown */
input:placeholder-shown {
  border: 2px dashed red;
  border-radius: 3px;
}
```

```html
<input type="text" placeholder="placeholder" />
```

> 위와 같은 상황에서 input에 placeholder가 보여지면, 빨간색의 점선 테두리가 생깁니다.
