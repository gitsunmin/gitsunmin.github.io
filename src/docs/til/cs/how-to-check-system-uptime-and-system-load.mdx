# How to check system uptime and system load

uptime 명령어는 시스템의 부하 정보와 시스템이 실행되고 있는 시간을 보여주는 명령어입니다. 일반적으로 다음과 같은 형태의 결과값을 반환합니다.

```shell
$ uptime
```
---
아래와 비슷한 내용으로 뜨게될 것입니다.
```shell
 0:10  up 10 mins, 2 users, load averages: 2.43 5.77 5.94
```

0:10 : 현재 시간을 나타냅니다.   

up 10 mins : 시스템이 실행된 시간을 나타냅니다. 이 경우에는 시스템을 실행한지 10분이 지난 후   

2 users : 현재 시스템에 로그인된 사용자 수를 나타냅니다. (터미널 수)   

load averages: 2.43 5.77 5.94 : 시스템 부하 정보를 나타냅니다. 이 경우에는 1분, 5분, 15분 동안의 평균 부하를 각각 2.43 5.77 5.94로 보여줍니다. 이 값은 시스템이 현재 처리하는 작업의 양을 나타내며, 일반적으로 1을 넘어서는 경우에는 시스템이 부하 상태에 있는 것으로 판단할 수 있습니다.
