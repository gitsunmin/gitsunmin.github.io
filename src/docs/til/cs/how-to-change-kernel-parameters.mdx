# How to change kernel parameters
sysctl을 서용하여 모든 매개변수를 조회. (mac or linux에서 사용.)

```shell
$ sysctl -a
```
---
```shell
$ sysctl ${매개변수_이름}
```
---

그리고 위험한 명령어 이지만, 커널 매개변수를 영구히 변경하는 방법이 있습니다.

---

```shell
$ sudo su
# 관리자 권한을 얻는다.

$ sysctl -w <TUNABLE_CLASS>.<PARAMETER>=<TARGET_VALUE> >> /etc/sysctl.conf
# sysctl -w를 이용하여 매개변수 값을 변경하고, /etc/sysctl.conf파일에 영구적으로 덮어쓰기합니다.
```
