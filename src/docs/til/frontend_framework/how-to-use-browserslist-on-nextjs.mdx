# How to use browserslist on Next.js

Next.js에서는 특정 브라우저와 버전에 대해서 지원을 제한하는 browserslist을 사용할 수 있습니다.

`package.json` browserslist은 아래와 같이 정의할 수 있습니다.

```json
{
  "browserslist": {
    "production": [
      ">0.2%",
      "not dead",
      "not op_mini all"
    ],
    "development": [
      "last 1 chrome version",
      "last 1 firefox version",
      "last 1 safari version"
    ]
  }
}
```

이런식으로 정의를 해두면, Next.js에서는 빌드 시 브라우저에 맞는 Polyfills를 제공해줍니다.

해당 내용은 Next.js의 공식 홈페이지에서도 자세히 나와있기 때문에 [링크](https://nextjs.org/docs/14/architecture/supported-browsers)에서 확인할 있습니다.
