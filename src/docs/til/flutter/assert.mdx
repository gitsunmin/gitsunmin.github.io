# Assert

Flutter(Dart)에서 Assert는 조건이 거진인 경우에 예외를 발생시키는 함수입니다. 
> 참고) 개발 모드인 Debug 모드에서만 동작하도록 설계되었으며, Release 모드에서는 동작하지 않습니다.

## 사용법

```dart
assert(조건, "예외 메시지");
assert(a == b); // 메시지 생략 가능

```

## 예제

```dart
void main() {
  int a = 10;
  int b = 20;
  assert(a == b, "a와 b는 같아야 합니다.");
}
```
