# about tauri

Tauri는 Electron과 비슷한 Desktop Application을 빌드하는 도구이며, HTML, CSS, JAVASCRIPT를 이용하여 개발 가능합니다. Electron과의 차이점은 Rust를 이용개발 개발되었고, CLI는 node로 사용가능합니다. 그리고 실행시간, 번들 크기와 메모리 사용량 등 성능면에서 Electron보다 이점이 있습니다. 다만, 2019년 12월에 pre-Alpha 출시를 한 만큼 아직 커뮤니티의 규모가 작은 상태입니다. 앞으로는 mobile app도 지원 예정이라고 하니, 성장 가능성이 충분해 보입니다.
