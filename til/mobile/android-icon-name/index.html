Android Icon Name

안드로이드 개발에서 아이콘으로 사용되는 이미지들은 다음과 같다.

크기로 구분하자면,

  • mipmap-ldpi(Low Density) : 대략 120dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 36x36
  • mipmap-mdpi(Medium Density) : 대략 160dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 48x48
  • mipmap-hdpi(High Density) : 대략 240dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 72x72
  • mipmap-xhdpi(Xtra High Density) : 대략 320dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 96x96
  • mipmap-xxhdpi(XXtra High Density) : 대략 480dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 144x144
  • mipmap-xxxhdpi(XXXtra High Density) : 대략 640dpi(dots per inch) 정도의 해상도를 가진 기기에 사용되는 아이콘, 크기는 192x192
  • mipmap-anydpi-v26 : 안드로이드 8.0(Oreo) 이상에서 사용되는 아이콘, 크기는 512x512

이렇게 구분할 수 있습니다.

그리고 안드로이드 프로젝트의 res 폴더들은 minimap 뿐만아니라 drawable로도 구분할 수 있습니다.

이 drawable은 아이콘 뿐만 아니라, 이미지 리소스들을 모아놓는 폴더입니다. (예를들면, splash 이미지 등)
