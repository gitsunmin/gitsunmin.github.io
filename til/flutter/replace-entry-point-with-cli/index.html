Replace Entry point with CLI

Flutter에서 기본 Entry point는 lib/main.dart 입니다.

하지만 특수한 목적에 따라서 entry point를 변경하여 Flutter를 실행 시키는 방법이 있습니다.

flutter run -t ./lib/main.dev.dart

위 명령어를 사용하면 (-t : target) Entry point를 변경하여 Flutter를 실행시킬 수 있습니다.

저는 개발환경에서 다른 동작을 위해서 사용하였습니다.
