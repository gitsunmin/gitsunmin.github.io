Flexbox

flexboxлҠ” л·°нҸ¬нҠёлӮҳ мҡ”мҶҢмқҳ нҒ¬кё°к°Җ л¶ҲлӘ…нҷ•н•ҳкұ°лӮҳ лҸҷм ҒмңјлЎң ліҖн•  л•Ңм—җлҸ„ нҡЁмңЁм ҒмңјлЎң мҡ”мҶҢлҘј л°°м№ҳ, м •л ¬, л¶„мӮ°н•  мҲҳ мһҲлҠ” л°©лІ•мқ„ м ңкіөн•ҳлҠ” CSS3мқҳ мғҲлЎңмҡҙ л Ҳмқҙм•„мӣғ л°©мӢқмқҙлӢӨ. flexboxмқҳ мһҘм җмқ„ н•ң л§Ҳл””лЎң н‘ңнҳ„н•ҳл©ҙ вҖҳліөмһЎн•ң кі„мӮ° м—Ҷмқҙ мҡ”мҶҢмқҳ нҒ¬кё°мҷҖ мҲңм„ңлҘј мң м—°н•ҳкІҢ л°°м№ҳн•  мҲҳ мһҲлӢӨвҖҷлқјкі  н•  мҲҳ мһҲлӢӨ. м •л ¬, л°©н–Ҙ, мҲңм„ң, нҒ¬кё° л“ұмқ„ мң м—°н•ҳкІҢ мЎ°м Ҳн•  мҲҳ мһҲкё° л•Ңл¬ём—җ лі„лҸ„мқҳ л¶„кё° мІҳлҰ¬лҘј мӨ„мқј мҲҳ мһҲкі , CSSл§ҢмңјлЎң лӢӨм–‘н•ң л Ҳмқҙм•„мӣғмқ„ кө¬нҳ„н•  мҲҳ мһҲлӢӨ.

Flex ContainerмҷҖ Flex Item

flexboxлҠ” л‘җ к°Җм§Җ к°ңл…җмқ„ кё°л°ҳмңјлЎң лҸҷмһ‘н•ңлӢӨ. л°”лЎң flex containerмҷҖ flex itemмқҙлӢӨ. flex containerлҠ” display мҶҚм„ұ к°’мңјлЎң flexлҘј к°–лҠ” мҡ”мҶҢлҘј л§җн•ҳл©°, flex itemмқҖ flex containerмқҳ мһҗмӢқ мҡ”мҶҢлҘј л§җн•ңлӢӨ.

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

мң„мқҳ мҪ”л“ңм—җм„ң .flex-containerлҠ” flex containerмқҙкі , .flex-itemмқҖ flex itemмқҙлӢӨ.

Flex Containerм—җ лҢҖн•ҙм„ң

Flex ContainerлҠ” Flex Layoutмқ„ кө¬нҳ„н•ҳкё° мң„н•ң мөңмғҒмң„ мҡ”мҶҢмқҙл©°, Flex Containerмқҳ мһҗмӢқ мҡ”мҶҢлЎң лӢӨмӢң Flex ContainerлҘј нҸ¬н•Ён•  мҲҳлҸ„ мһҲкё° л•Ңл¬ём—җ, мӣҗн•ҳлҠ” Layoutмқ„ кө¬нҳ„н•ҳлҠ” кІғмқҙ к°ҖлҠҘн•©лӢҲлӢӨ.

display

Flex ContainerлҘј м •мқҳн•ҳкё° мң„н•ҙм„ңлҠ” display мҶҚм„ұмқ„ мӮ¬мҡ©н•ҙм•ј н•©лӢҲлӢӨ. display мҶҚм„ұ к°’мңјлЎң flexлҘј мӮ¬мҡ©н•ҳл©ҙ Flex ContainerлЎң м •мқҳлҗ©лӢҲлӢӨ.

.flex-container {
  display: flex;
}

flex-direction

flex-direction мҶҚм„ұмқҖ Flex Containerмқҳ мЈјм¶•(main-axis)мқ„ м„Өм •н•©лӢҲлӢӨ. мЈјм¶•мқҖ Flex Itemмқ„ л°°м№ҳн•ҳлҠ” л°©н–Ҙмқ„ м„Өм •н•ҳлҠ”лҚ° мӮ¬мҡ©лҗ©лӢҲлӢӨ. кё°ліёк°’мқҖ rowмһ…лӢҲлӢӨ.

  • row: Flex Itemмқ„ мҲҳнҸүм¶•мңјлЎң л°°м№ҳн•©лӢҲлӢӨ.
  • row-reverse: rowмҷҖ к°ҷм§Җл§Ң, Flex Itemмқ„ л°ҳлҢҖ л°©н–ҘмңјлЎң л°°м№ҳн•©лӢҲлӢӨ.
  • column: Flex Itemмқ„ мҲҳм§Ғм¶•мңјлЎң л°°м№ҳн•©лӢҲлӢӨ.
  • column-reverse: columnкіј к°ҷм§Җл§Ң, Flex Itemмқ„ л°ҳлҢҖ л°©н–ҘмңјлЎң л°°м№ҳн•©лӢҲлӢӨ.
.flex-container {
  flex-direction: row;
}

flex-wrap

flex-wrap мҶҚм„ұмқҖ Flex Itemмқҙ Flex Containerмқҳ нҒ¬кё°м—җ л§һкІҢ мӨ„мқ„ л°”кҝҖм§Җ м—¬л¶ҖлҘј м„Өм •н•©лӢҲлӢӨ. кё°ліёк°’мқҖ nowrapмһ…лӢҲлӢӨ.

  • nowrap: Flex Itemмқ„ н•ң мӨ„м—җ л°°м№ҳн•©лӢҲлӢӨ.
  • wrap: Flex Itemмқ„ м—¬лҹ¬ мӨ„м—җ л°°м№ҳн•©лӢҲлӢӨ.
  • wrap-reverse: wrapкіј к°ҷм§Җл§Ң, Flex Itemмқ„ л°ҳлҢҖ л°©н–ҘмңјлЎң л°°м№ҳн•©лӢҲлӢӨ.
.flex-container {
  flex-wrap: wrap;
}

flex-flow

flex-flow мҶҚм„ұмқҖ flex-directionкіј flex-wrap мҶҚм„ұмқ„ н•ң лІҲм—җ м„Өм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

.flex-container {
  flex-flow: row wrap;
}

justify-content

justify-content мҶҚм„ұмқҖ Flex Itemмқ„ Flex Container лӮҙм—җм„ң м •л ¬н•ҳлҠ” л°©лІ•мқ„ м„Өм •н•©лӢҲлӢӨ. мЈјм¶•(main-axis)мқ„ кё°мӨҖмңјлЎң лҸҷмһ‘н•©лӢҲлӢӨ.

  • flex-start: Flex Itemмқ„ мӢңмһ‘ л¶Җл¶„м—җ м •л ¬н•©лӢҲлӢӨ.
  • flex-end: Flex Itemмқ„ лҒқ л¶Җл¶„м—җ м •л ¬н•©лӢҲлӢӨ.
  • center: Flex Itemмқ„ к°ҖмҡҙлҚ°м—җ м •л ¬н•©лӢҲлӢӨ.
  • space-between: мІ« лІҲм§ё Flex ItemмқҖ мӢңмһ‘ л¶Җл¶„м—җ, л§Ҳм§Җл§ү Flex ItemмқҖ лҒқ л¶Җл¶„м—җ м •л ¬лҗҳкі , лӮҳлЁём§Җ Flex ItemмқҖ мӮ¬мқҙм—җ кі лҘҙкІҢ м •л ¬лҗ©лӢҲлӢӨ.
  • space-around: лӘЁл“  Flex Itemмқ„ к· л“ұн•ң м—¬л°ұмқ„ нҸ¬н•Ён•ҳм—¬ м •л ¬н•©лӢҲлӢӨ.
.flex-container {
  justify-content: center;
}

align-items

align-items мҶҚм„ұмқҖ Flex Itemмқ„ Flex Container лӮҙм—җм„ң м •л ¬н•ҳлҠ” л°©лІ•мқ„ м„Өм •н•©лӢҲлӢӨ. көҗм°Ём¶•(cross-axis)мқ„ кё°мӨҖмңјлЎң лҸҷмһ‘н•©лӢҲлӢӨ.

  • flex-start: Flex Itemмқ„ мӢңмһ‘ л¶Җл¶„м—җ м •л ¬н•©лӢҲлӢӨ.
  • flex-end: Flex Itemмқ„ лҒқ л¶Җл¶„м—җ м •л ¬н•©лӢҲлӢӨ.
  • center: Flex Itemмқ„ к°ҖмҡҙлҚ°м—җ м •л ¬н•©лӢҲлӢӨ.
  • baseline: Flex Itemмқ„ л¬ёмһҗ кё°мӨҖм„ м—җ м •л ¬н•©лӢҲлӢӨ.
  • stretch: Flex Itemмқ„ көҗм°Ём¶•(cross-axis)м—җ л§һкІҢ лҠҳлҰҪлӢҲлӢӨ.
.flex-container {
  align-items: center;
}
