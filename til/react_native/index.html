Core Components and Native Components

React Nativeмқҳ UI м»ҙнҸ¬л„ҢнҠёлҠ” нҒ¬кІҢ Core ComponentмҷҖ Native ComponentлЎң кө¬л¶„лҗ©лӢҲлӢӨ. Core ComponentsлҠ” React Nativeм—җм„ң м ңкіөн•ҳлҠ” кё°ліё UI м»ҙнҸ¬л„ҢнҠёл“Өмқҙл©°, л№Ңл“ң мӢң к°Ғ нғҖк№ғм—җ л§һлҠ” Native ComponentлЎң ліҖнҷҳлҗ©лӢҲлӢӨ. мҰү, Native ComponentлҠ” к°Ғ н”Ңлһ«нҸјм—җм„ң мӮ¬мҡ©лҗҳлҠ” Swift, Java нҳ№мқҖ Kotlin мҪ”л“ңлЎң л§Ңл“Өм–ҙм§„ м»ҙнҸ¬л„ҢнҠёмһ…лӢҲлӢӨ. лҳҗн•ң, React Nativeм—җм„ңлҠ” н•„мҡ”м—җ л”°лқј л„ӨмқҙнӢ°лёҢ лӘЁл“Ҳмқ„ мһ‘м„ұн•ҳм—¬ Core ComponentмҷҖ нҶөн•©лҗң Native ComponentлҘј л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.

м»ҙнҸ¬л„ҢнҠёмқҳ кө¬мЎ°лҠ” м•„лһҳмҷҖ к°ҷмқҙ н‘ңнҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Platform-Specific Components

FlutterлҘј н•ҳл©ҙм„ң лҠҗлӮҖ кІғмқҖ, вҖңMulti PlatformмқёлҚ°лҸ„ л¶Ҳкіјн•ҳкі , л¶„кё°лҘј кі„мҶҚ міҗмЈјлҠ” кІғмқҙ л¶ҲнҺён•ҳлӢӨ.вҖқ мҳҖмҠөлӢҲлӢӨ. к·ёлҹ°лҚ°, React Nativeм—җм„ңлҠ” н•ҙлӢ№ Classк°Җ м •л§җ мң мҡ©н•ҳкІҢ мӮ¬мҡ©лҗ  кІғ к°ҷлӢӨлҠ” мғқк°Ғмқҙ л“Өм—ҲмҠөлӢҲлӢӨ.

  height: Platform.OS === 'ios' ? 200 : 100,

мқҙлҹ°мӢқмңјлЎң OSлі„лЎң лӢӨлҘё Style к°қмІҙлҘј л§Ңл“Ө мҲҳ мһҲмқ„ лҝҗл§Ңм•„лӢҲлқј, Select MethodлҘј мӮ¬мҡ©н•ҳл©ҙ,

    flex: 1,
    ...Platform.select({
      ios: {
        backgroundColor: 'red',
      },
      android: {
        backgroundColor: 'green',
      },
      default: {
        // other platforms, web for example
        backgroundColor: 'blue',
      },
    }),
  },
});

TextInput

TextInputмқҖ мӮ¬мҡ©мһҗк°Җ н…ҚмҠӨнҠёлҘј мһ…л Ҙн•  мҲҳ мһҲлҠ” м»ҙнҸ¬л„ҢнҠёмһ…лӢҲлӢӨ.

      style={{ height: 40, borderColor: 'gray', borderWidth: 1 }}

FlatList

FlatListлҠ” л§ҺмқҖ лҚ°мқҙн„° н•ӯлӘ©мқ„ нҡЁмңЁм ҒмңјлЎң л ҢлҚ”л§Ғн•ҳлҠ” м»ҙнҸ¬л„ҢнҠёлЎң, к°ҖмғҒнҷ”лҗң лӘ©лЎқмқ„ мғқм„ұн•©лӢҲлӢӨ.

import { FlatList, Text } from 'react-native';

const App = () => (
  <FlatList
    data={data}
    renderItem={({ item }) => <Text>{item.value}</Text>}
    keyExtractor={item => item.key}
  />
);

    },

    body: 'Your Body goes here',
  }}
  style={{width: 400, height: 400}}
/>

uriлҘј м„Өм •н•  л•Ңм—җ, к°„лӢЁн•ң Network м„Өм •л“Өмқ„ мЎ°мһ‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

м°ёкі лЎң cache м •мұ…мқҖ м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ. default: Use the native platforms default strategy. reload: The data for the URL will be loaded from the originating source. No existing cache data should be used to satisfy a URL load request. force-cache: The existing cached data will be used to satisfy the request, regardless of its age or expiration date. If there is no existing data in the cache corresponding the request, the data is loaded from the originating source. only-if-cached: The existing cache data will be used to satisfy a request, regardless of its age or expiration date. If there is no existing data in the cache corresponding to a URL load request, no attempt is made to load the data from the originating source, and the load is considered to have failed.
