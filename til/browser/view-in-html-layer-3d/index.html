View in HTML Layer 3D

Google Chrome에서 현재 화면의 레이어를 3D로 표시한 것을 보는 방법이 있습니다.

  • Chrome 개발자 도구를 킨다.
  • 오른쪽 상단에 더보기 표시를 클릭하여 메뉴를 확인한다.
  • More Tools를 클릭한다.
  • Layers를 클릭한다.
  • 기본 옵션으로는 상하좌우로 움직일 수 있는 모드이지만, Layers 메뉴의 상단에 회전 아이콘을 클릭한다.
  • Layers 메뉴의 화면을 드래그 해보면 Layers가 회전하는 것을 볼 수 있다.

z-index를 확인하거나, 웹 페이지의 전체 모습을 확인할 때 유용하게 사용할 수 있을 것 같다.
