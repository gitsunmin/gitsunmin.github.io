м„ңлЎ 

WebAssemblyлҠ” V8 м—”м§„м—җм„ң мӢӨн–үн•  мҲҳ мһҲлҠ” Javascript мҷёмқҳ нҸ¬л§·мһ…лӢҲлӢӨ. мқҙ нҸ¬л§·мқҖ Javascript ліҙлӢӨ лҚ” м ҖмҲҳмӨҖ м–ём–ҙм—җ к°Җк№қкі  нҸүк· м ҒмңјлЎң лҚ” лӮҳмқҖ м„ұлҠҘмңјлЎң мӢӨн–үлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёл ҮлӢӨл©ҙ, м–ҙл– н•ң м°Ёмқҙ л•Ңл¬ём—җ мқҙлҹ¬н•ң кІ°кіјк°Җ лҸ„м¶ңлҗҳм—ҲлҠ”м§Җ м•Ңм•„ліҙкі , мӮ¬мҡ© л°©лІ•лҸ„ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.

ліёлЎ 

V8 м—”м§„мқҖ л¬ҙм—Үмқёк°Җ?

м„ңлЎ м—җм„ң м–ёкёүн–Ҳл“Ҝмқҙ, WebAssemblyлҠ” V8 м—”м§„м—җм„ң мӢӨн–үлҗҳлҠ” м ҖмҲҳмӨҖ нҸ¬л§·мһ…лӢҲлӢӨ. к·ёл ҮлӢӨл©ҙ, WebAssemblyлҘј м•Ңкё° м „м—җ V8 м—”м§„мқҖ л¬ҙм—Үмқјк№Ңмҡ”? к°„лӢЁн•ҳкІҢлқјлҸ„ мЎ°мӮ¬н•ҳкі  л„ҳм–ҙк°Җм•јкІ лӢӨлҠ” мғқк°Ғмқҙ л“Өм–ҙм„ң м°ҫм•„ліҙм•ҳмҠөлӢҲлӢӨ.

V8 м—”м§„мқҖ Googleм—җм„ң л§Ңл“  мҳӨн”ҲмҶҢмҠӨ н”„лЎңм қнҠёмқҙл©° JavascriptмҷҖ WebAssemblyлҘј мӢӨн–үн•  мҲҳ мһҲлҠ” м—”м§„мһ…лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ, JavascriptлҠ” л¶„м„қлҗҳм–ҙ кё°кі„м–ҙ(Machine Code)лЎң ліҖнҷҳлҗҳл©°, мқҙлҘј нҶөн•ҙ м»ҙн“Ён„°мқҳ CPUк°Җ н•ҙлӢ№ лӘ…л №мқ„ мҲҳн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лҚ” мһҗм„ён•ң лӮҙмҡ©мқҙ к¶ҒкёҲн•ҳмӢңлӢӨл©ҙ, м•„лһҳмқҳ л§ҒнҒ¬м—җм„ң мўҖ лҚ” м°ҫм•„ліҙл©ҙ лҸ„мӣҖмқҙ лҗ  кІғ к°ҷмҠөлӢҲлӢӨ. Link

V8м—”м§„м—җм„ңмқҳ Javascript

мқҙ кёҖмқҳ мЈјм ңлҠ” WebAssemblyмқҙм§Җл§Ң, WebAssemblyмқҳ лҸҷмһ‘ л°©мӢқмқ„ м•Ңкё° м „м—җ Javascriptк°Җ V8м—”м§„м—җм„ң м–ҙл–»кІҢ лҸҷмһ‘н•ҳлҠ”м§Җ к°„лӢЁнһҲ м•Ңм•„ліҙкі , WebAssemblyмқҳ лҸҷмһ‘мқ„ мӮҙнҺҙліҙлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ.

м•„лһҳмқҳ мқҙлҜём§ҖлҠ” Linkм—җм„ң к°Җм§Җкі мҳЁ V8м—”м§„м—җм„ңмқҳ Javascriptмқҳ лҸҷмһ‘ мҲңм„ңлҘј к·ёл ӨлӮё кІғмһ…лӢҲлӢӨ.

мһҗл°”мҠӨнҒ¬лҰҪнҠё мҶҢмҠӨ мҪ”л“ңлҠ” кө¬мІҙм Ғмқё л¬ёлІ• к·ңм№ҷмқ„ л”°лҘҙлҠ” н…ҚмҠӨнҠё нҳ•нғңлЎң мЎҙмһ¬н•©лӢҲлӢӨ. мқҙлҹ¬н•ң н…ҚмҠӨнҠёлҠ” л¬ёлІ•м Ғ л¶„м„қмқ„ нҶөн•ҙ кё°кі„к°Җ мқҙн•ҙн•  мҲҳ мһҲлҠ” нҳ•нғңлЎң лІҲм—ӯлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ лІҲм—ӯ кіјм •мқ„ нҡЁмңЁм ҒмңјлЎң мҲҳн–үн•ҳкё° мң„н•ҙ, мҶҢмҠӨ мҪ”л“ңлҠ” лЁјм Җ кө¬мЎ°нҷ”лҗң лҚ°мқҙн„° нҳ•нғңмқё м¶”мғҒ кө¬л¬ё нҠёлҰ¬(Abstract Syntax Tree, AST)лЎң ліҖнҷҳлҗ©лӢҲлӢӨ.

Googleмқҳ V8 м—”м§„мқҖ мһҗл°”мҠӨнҒ¬лҰҪнҠё мҪ”л“ңлҘј мІҳлҰ¬н•ҳкё° мң„н•ҙ нҢҢм„ң(Parser)лҘј мӮ¬мҡ©н•ҳм—¬ мқҙлҹ¬н•ң ASTлҘј мғқм„ұн•©лӢҲлӢӨ. ASTлҠ” мҶҢмҠӨ мҪ”л“ңмқҳ кө¬л¬ём Ғ кө¬мЎ°лҘј л°ҳмҳҒн•ҳл©°, к°Ғ л…ёл“ңлҠ” к°ңлі„ мҪ”л“ң кө¬м„ұ мҡ”мҶҢ(мҳҲ: ліҖмҲҳ м„ м–ё, н•ЁмҲҳ нҳём¶ң л“ұ)лҘј лӮҳнғҖлғ…лӢҲлӢӨ. мғқм„ұлҗң ASTлҠ” мҪ”л“ңмқҳ мқҳлҜёлҘј л¶„м„қн•ҳкі  мӢӨн–ү кі„нҡҚмқ„ м„ёмҡ°лҠ” кё°л°ҳмңјлЎң мӮ¬мҡ©лҗ©лӢҲлӢӨ.

V8 м—”м§„м—җ лӮҙмһҘлҗң IgnitionмқҙлқјлҠ” мқён„°н”„лҰ¬н„°лҠ” мқҙ ASTлҘј н•ҙм„қн•ҳм—¬ л°”мқҙнҠёмҪ”л“ңлЎң ліҖнҷҳн•ҳлҠ” м—ӯн• мқ„ мҲҳн–үн•©лӢҲлӢӨ. л°”мқҙнҠёмҪ”л“ңлҠ” V8 м—”м§„мқҳ м ҖмҲҳмӨҖ м–ём–ҙлЎң, лҚ” л№ лҘё мӢӨн–үмқ„ к°ҖлҠҘн•ҳкІҢ н•ҳлҠ” мӨ‘к°„ н‘ңнҳ„ нҳ•нғңмһ…лӢҲлӢӨ. мқҙ л°”мқҙнҠёмҪ”л“ңлҠ” TurboFanмқҙлқјкі  л¶ҲлҰ¬лҠ” кі кёү мөңм Ғнҷ” м»ҙнҢҢмқјлҹ¬м—җ мқҳн•ҙ мІҳлҰ¬лҗ©лӢҲлӢӨ.

TurboFanмқҖ мӢӨн–ү мӨ‘м—җ мҲҳм§‘лҗң м„ұлҠҘ лҚ°мқҙн„°лҘј л¶„м„қн•ҳкі , л°”мқҙнҠёмҪ”л“ңлҘј мөңм Ғнҷ”лҗң кё°кі„м–ҙ мҪ”л“ңлЎң ліҖнҷҳн•©лӢҲлӢӨ. мқҙ кіјм •мқҖ мҪ”л“ңмқҳ мӢӨн–ү мҶҚлҸ„лҘј нҒ¬кІҢ н–ҘмғҒмӢңнӮ¬ мҲҳ мһҲмңјл©°, мһҗл°”мҠӨнҒ¬лҰҪнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ м „л°ҳм Ғмқё м„ұлҠҘмқ„ мөңм Ғнҷ”н•ҳлҠ” лҚ° мӨ‘мҡ”н•ң м—ӯн• мқ„ н•©лӢҲлӢӨ.

лӢЁ, л№Ёк°„мғү нҷ”мӮҙн‘ңмқҳ лӮҙмҡ©мқҙ лӢӨмӢң BytecodeлЎң ліҖнҷҳлҗҳм–ҙм§ҖлҠ” кІғмқҙлқјкі  мҳӨн•ҙн•  мҲҳ мһҲм§Җл§Ң, к·ёлҹ° мқҳлҜёк°Җ м•„лӢҲлқј, лӢӨмқҢ лҸҷмһ‘мқ„ н•ҳкё° мң„н•ҙм„ң лӢӨмӢң IgnitionмңјлЎң лҸҢм•„к°„лӢӨкі  мқҙн•ҙн•ҳл©ҙ мўӢмқ„ кІғ к°ҷмҠөлӢҲлӢӨ. м»ҙнҢҢмқјмқҳ мөңмў… кІ°кіјл¬јмқҳ нҳ•нғңлҠ” кё°кі„к°Җ н•ҙм„қн•  мҲҳ мһҲлҠ” кё°кі„м–ҙ(Machin Code)к°Җ л§һмҠөлӢҲлӢӨ.

v8м—”м§„м—җм„ңмқҳ WebAssembly

V8 м—”м§„м—җм„ң JavascriptлҠ” мҶҢмҠӨ мҪ”л“ңлҘј л¶„м„қн•ҳкі , AST(Abstract Syntax Tree) ліҖнҷҳмқ„ кұ°міҗ, Ignition мқён„°н”„лҰ¬н„°лҘј нҶөн•ҙ л°”мқҙнҠё мҪ”л“ңлЎң ліҖнҷҳлҗ©лӢҲлӢӨ. л°ҳл©ҙ, WebAssemblyлҠ” мқҙм§„ мҪ”л“ң нҳ•нғңлЎң лёҢлқјмҡ°м Җм—җ м „лӢ¬лҗҳл©°, к°„лӢЁн•ҳкі  л№ лҘё нҢҢмӢұ нӣ„м—җ л°”лЎң кё°кі„м–ҙлЎң м»ҙнҢҢмқјмқҙ к°ҖлҠҘн•ң мғҒнғңк°Җ лҗ©лӢҲлӢӨ. мқҙ кіјм •мқ„ лӢҙлӢ№н•ҳлҠ” V8 м—”м§„мқҳ м»ҙнҢҢмқјлҹ¬лҠ” Liftoffлқјкі  л¶ҲлҰҪлӢҲлӢӨ. LiftoffлҠ” WebAssemblyлҘј л°”лЎң мӢӨн–ү к°ҖлҠҘн•ң кё°кі„м–ҙлЎң л№ лҘҙкІҢ ліҖнҷҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ. м„ұлҠҘ мөңм Ғнҷ”к°Җ н•„мҡ”н•ң кІҪмҡ°, м¶”к°Җм Ғмқё мөңм Ғнҷ”лҘј мң„н•ҙ TurboFan м»ҙнҢҢмқјлҹ¬к°Җ мӮ¬мҡ©лҗ©лӢҲлӢӨ.

LiftoffлҠ” WebAssemblyмқҳ л№ лҘё м»ҙнҢҢмқјмқ„ мұ…мһ„м§Җл©°, мөңм Ғнҷ”к°Җ н•„мҡ”н•ң кІҪмҡ° TurboFanмңјлЎң мҪ”л“ңлҘј л„ҳкІЁ лҚ” к№ҠмқҖ мөңм Ғнҷ”лҘј мҲҳн–үн•©лӢҲлӢӨ. мқҙ л‘җ м»ҙнҢҢмқјлҹ¬мқҳ нҳ‘л Ҙмқ„ нҶөн•ҙ V8 м—”м§„мқҖ WebAssembly мҪ”л“ңлҘј нҡЁмңЁм ҒмңјлЎң мІҳлҰ¬н•ҳл©°, л№ лҘё мӢңмһ‘кіј кі м„ұлҠҘ мң м§ҖлҘј лҸҷмӢңм—җ лӢ¬м„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

#[no_mangle]
pub extern "C" fn fibonacci(n: usize) -> u64 {
    if n == 0 {
        return 0;
    }

    let mut a = 0;
    let mut b = 1;
    for _ in 1..n {
        let temp = a + b;
        a = b;
        b = temp;
    }

    a
}
